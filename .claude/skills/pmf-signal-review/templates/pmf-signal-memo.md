## Run /pmf-signal-review with this memo

Save this file and run `/pmf-signal-review`. Claude will assess your signals,
give a verdict (Strong / Mixed / Weak), and recommend what to do next.

**Bring honesty.** The skill works best when you are not trying to put a positive
spin on weak signals. If the data is weak, name it. The recommendation will be
more useful.
